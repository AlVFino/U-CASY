U-CASY adalah aplikasi kasir berbasis HTML, CSS, dan JavaScript yang digunakan untuk mengelola transaksi layanan Unit Produksi Jurusan (UPJ). Aplikasi ini mencakup sistem kasir, data barang/jasa, pencatatan piutang pelanggan, serta laporan transaksi, semuanya dalam tampilan yang modern, responsif, dan mudah digunakan.

**Languages**
- HTML
- CSS
- JavaScript

**Fitur**
- Sistem kasir dengan input cepat, keranjang dinamis, dan total otomatis
- Pembayaran cash & piutang
- Manajemen data barang/jasa (tambah, edit, hapus)
- Manajemen pelanggan piutang & riwayatnya
- Dashboard ringkasan transaksi dan data UPJ
- Laporan transaksi dengan filter tanggal + ekspor (CSV/PDF)
- Navbar dinamis & antarmuka responsif

**Instalasi**
1. Clone atau download repository ini
2. Buka folder project menggunakan VS Code
3. (Opsional) Jalankan menggunakan Live Server untuk pengalaman yang lebih optimal, Atau cukup buka run index.html di root project

**Penggunaan**
1. Buka /pages/index.html
2. Navigasikan halaman melalui navbar:
    - Dashboard
    - Kasir
     - Barang/Jasa
    - Pelanggan (Piutang)
    - Laporan
